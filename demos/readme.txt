1. PCLdemo_model
- Visual studio demo for processing L3D data files and modelling surfaces.
- Use this demo to display L3D data, model surfaces, and save as GEO files.

2. PCLdemo_segment
- Visual studio demo for segmenting surfaces.
- Use this demo to load GEO files and segment surfaces.

3. PCLdemo_match
- Visual studio demo for matching surfaces.
- Use this demo to test surface matching algorithms after segmenting surfaces.

4. PCLdemo_registration
- Visual studio demo for surface registration.


Note: Please insatll and config PCL1.13 for these demos.
